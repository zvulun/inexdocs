Driver Face Capture (DFC) Quick Start Guide

Click this Driver Face Capture Solution link for additional documentation.

You will need:

  • Laptop with Windows 10 or above with Windows Feature NET 4.5 enabled.

  • Chrome or Microsoft Edge browser.

Checklist

  • Prepare components and tools

  • Plan your site

  • Prepare cables

  • Install Camera System(s) and other components

  • Connect components (wiring)

  • Power up and set up IP

  • Configure LPR/OV camera settings

  • Aim and calibrate

  • Verify system operation

 — If you are using a DIN rail-mount power supply, you will need to cut off the rings at the ends of the Camera System’s power connections, and attach pins in their place.
 — After mounting, remove the protective film from the Camera System’s front window.
 — You can use IZ Discovery to find the desired Camera System and change its network settings. Defaults are 192.168.5.110 (onboard DPU processor), 192.168.5.111 (LPR camera) and 192.168.5.112 (OV camera).
