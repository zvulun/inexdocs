IZA500G Hardware Revision History

ALPR All-in-one AI Camera System

Document Description

Hardware revision histories (HRH) inform our customers about changes being made to our Inex Technologies products, such as physical appearance, hardware improvements, changes to package contents, part number changes, etc. Firmware changes are not included.

Q3-2023 Revision F3

Change Summary

Summary

  • Input voltage now 24VDC or 48VDC (for new P48 models); not 12-24 VDC as was previously

  • New power input option of PoE+ (for new P48 models; requires IZxPOE Gigabit PoE++ Injector)

  • LPR Capture Distance changed from 5-70 ft (1.5-21 m) to:
    S - Short distance, 5-35 ft (1.5-11 m)
    L - Long distance, 30-70 ft (9-21 m)

  • Model number additions to accommodate new power options

Product release date

Q3-2023
